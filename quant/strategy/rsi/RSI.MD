## RSI指标概述

### 计算逻辑

`RSI`（Relative Strength Index，相对强弱指数）是由J. Welles Wilder Jr.在1978年开发的技术分析指标。

**基本计算步骤：**

1. **计算价格变化**：
   - 计算每个周期的价格变化（当前收盘价 - 前一收盘价）
   - 将正数变化记为`U`（上涨），负数变化取绝对值记为`D`（下跌）

2. **计算平均值**：
   - 计算n个周期内上涨幅度的平均值：`AvgU = SUM(U, n) / n`
   - 计算n个周期内下跌幅度的平均值：`AvgD = SUM(D, n) / n`

3. **计算相对强度**：
   - `RS = AvgU / AvgD`

4. **计算RSI**：
   - `RSI = 100 - (100 / (1 + RS))`

在实际应用中，通常使用指数移动平均来平滑计算：
- `AvgU = (AvgU_prev * (n-1) + U_current) / n`
- `AvgD = (AvgD_prev * (n-1) + D_current) / n`

### 背景和理论基础

- **动量指标**：`RSI`属于动量震荡指标，用于衡量价格变动的速度和变化
- **范围限制**：`RSI`值域在0到100之间
- **市场情绪反映**：通过比较上涨和下跌幅度来反映市场情绪

### 使用场景

1. **超买超卖判断**：
   - `RSI > 70`：市场处于超买状态，可能出现回调
   - `RSI < 30`：市场处于超卖状态，可能出现反弹

2. **背离信号**：
   - **顶背离**：价格创新高但`RSI`未创新高，预示可能下跌
   - **底背离**：价格创新低但`RSI`未创新低，预示可能上涨

3. **中线趋势判断**：
   - `RSI`在50以上表示强势市场
   - `RSI`在50以下表示弱势市场

4. **支撑阻力识别**：
   - `RSI`的历史高低点可作为未来支撑阻力参考

### 注意事项

- 不同市场和资产可能需要调整超买超卖阈值
- 在强烈趋势市场中，`RSI`可能长时间处于超买或超卖区域
- 应结合其他技术指标和基本面分析使用，避免单一指标决策
